# Galaktion v0.1.1

## Λίγα λόγια
Η γλώσσα Galaktion είναι μια scripting γλώσσα ειδικού σκοπού, η οποία έχει ως στόχο να διευκολύνει την διαχείριση δεδομένων XML σε μεμονομένα αρχεία αλλά και σε δομές συσχετισμένων μεταξύ τους αρχείων XML.

## Εγκατάσταση
Για να εγκαταστήσετε την γλώσσα στον υπολογιστή σας που τρέχει Windows, απλώς κατεβάστε τον installer από αυτήν την σελίδα και εκτελέστε τον.
Ο installer θα εγκαταστήσει τον διερμηνέα της γλώσσας (Galaktion.exe) και θα τον προσθέσει στην μεταβλητή περιβάλλοντος "Path".

## Χρήση
Μετά την εγκατάσταση θα πρέπει να μπορείτε να τρέξετε την γλώσσα απλώς γράφοντας στο terminal "Galaktion".
Αφού γράψουμε τον κώδικά μας σε ένα αρχείο __.gal__ μπορούμε να τον εκτελέσουμε γράφοντας `Galaktion <όνομα αρχείου> [enter]`.
